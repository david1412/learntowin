Linux Commands 
1)pwd = It prints the path of the working directory, starting from the roots
2)echo =Used to display line of text/string that are passed as an argument
3)cd  = change directoty
4)touch= Used to create, change and modify timestamps of a file
5)mv=Used to move one or more files or directories from one place to another in file system like UNIX
6)escp+:1,$d=Deleting all the lines from the file
7)dd= Deleting line to line
8)dD= Deleting the paragraph
9):q!= Exit of vi > file
10)rm -r= Remove of directory
11)rm = Remove text file
