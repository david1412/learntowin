Linux commands : 
 esc :gg = move cursor to the start of the document
 shift +i = insert before cursor
 esc, :wq or :q = save file and close
 esc :1,$d = deletes all lines
 dd or :d = Delete current line
 vi <file-name> = creates and open a file
 pwd = present working directory
 cd <file name> = create directory
 cd .. = go up one folder level
 esc = terminate insert command
