 Linux commands : 
 - esc :gg = move cursor to the start of the fie
 - shift +i = insert before cursor ( to insert text in windows gitbash)
 - esc, :wq or :q = saves and closes file 
 - esc :1,$d = deletes all lines in the current open file
 - dd or :d = Delete current line where the cursor is positioned
 - vi <file-name> = creates and open a file for editing
 - pwd = present working directory (to check the present directory we are working)
 - cd <file name> = create directory to create files
 - cd .. = go up one folder level from the current directory
 - esc = terminate insert command and exit file editor
