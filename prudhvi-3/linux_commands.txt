               Here are few Linux commands
1)rm filename      =) 	Deletes a file
2)man              =)	Gives help information on a command
3)history          =)	Gives a list of all past commands typed in the current terminal session
4)clear            =)	Clears the terminal
5)i                =)	Insert at cursor (goes into insert mode)
6)a                =)	Write after cursor (goes into insert mode
7)ESC              =)	Terminate insert mode
8)dd               =)	Delete line
9)dw               =)	Delete word
10)dG              =)   Deleting lines from begining of cursor position
11)ESC:$d          =)   Deleting all matter
12):q!             =)   Exit of vi file
